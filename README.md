# vue-movie

### 使用方式

- `git clone`项目至本地
- `npm install`安装依赖
- `npm run serve`启动项目即可

### 技术栈

- `vue`
- `vuex`
- `vue-router`
- `better-scroll`
- `proxy`
- `axios`
